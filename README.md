# COVID-19 Data Analysis & Prediction 📊🦠

This project focuses on analyzing COVID-19 data to understand trends, patterns, and predictions using **Python**, **Machine Learning models**, and **Tableau visualizations**.

---

## 🔍 Project Overview
The goal of this project is to:
- Analyze the spread and impact of COVID-19
- Perform data cleaning and exploratory data analysis (EDA)
- Build machine learning models for prediction
- Create interactive dashboards using Tableau

---

## 🚀 Features
- COVID-19 data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Data visualization using Python and Tableau
- Machine Learning models for trend/prediction analysis
- Insightful dashboards for better decision-making

---

## 🧰 Technologies Used
- **Python**
- **Pandas, NumPy**
- **Matplotlib, Seaborn**
- **Scikit-learn**
- **Tableau**
- **Jupyter Notebook**
